Find the closest shops and services

All the functionalities that need to find the closest shops and services are implemented. Users will be able to filter shops and services mainly using two methods.

Category Based Filtering

Shops can be filtered using multiple categories such as food, grocery, medicine and many Google Maps related features such as zoom in/out to locations, get directions and rotating the map are implemented.

Location based filtering

Users can select the location using two methods. One is to use their current location and the other method is to use a custom location. A slider to change the range of filtering and find the closest shops is implemented.

App Features

All the features need to find the best shops such as nearest shops, cheapest shops with the highest quality etc, are implemented

MAINTAIN USER PROFILES

Users can create seller profiles or buyer profiles with necessary details including their contact information

NOTIFICATIONS

All those generated notifications can be found on the 'Activity Feed'. Here the possible notification types are push notifications, follower notifications, chat notifications, post like and comment notifications

CHAT FACILITY

All the basic chat functionalities are available for both sellers and buyers

CREATE POSTS

Sellers can create posts using the images taken from the the camera or using the images in the gallery. They can optionally add image descriptions and image locations to the each post

TIMELINE

Buyers can follow shops, receive favourite shop posts and rate and comment posts

SEARCH SHOPS

Users can search registered shops by their names. Using those search results users can navigate to shop pages and follow them easily

Choose Your Subscription Plan

All the buyers can use the app for no cost. A flexible in-app advertising payment mechanism is introduced to the shopkeepers according to their needs such as customer’s location, customer’s age, customer’s gender, number of target customers etc.
